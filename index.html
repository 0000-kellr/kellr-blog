Deine smarte Vorratskammer

Nie wieder Essen wegwerfen, nie wieder doppelt kaufen. Kellr trackt Vorräte, warnt vor Ablauf, bestellt nach — für die ganze Familie.

Was Kellr kann

📦

Vorräte scannen & tracken

Barcode scannen, fertig. Automatische Produkterkennung, Lagerorte, Kategorien. Alles auf einen Blick.

Ablauf-Warnungen

Gelb, orange, rot — Kellr warnt rechtzeitig bevor etwas abläuft. Kein Essen mehr wegwerfen.

Über das Projekt

🧑‍💻 Wer ist Michael?

IT-Profi aus der Microsoft-Welt. PowerShell, Azure, Automatisierung. Versteht Programmier-Grundlagen, aber hat null Erfahrung mit iOS oder SwiftUI. Kein Developer — ein Automatisierer, der KI auf die Probe stellt.

🔨 Tech Stack

  • AI: Claude Opus (Anthropic)
  • Platform: OpenClaw
  • Input: WhatsApp Voice Messages
  • App: SwiftUI / iOS
  • Backend: Supabase (Frankfurt, EU)
  • CI/CD: GitHub Actions
